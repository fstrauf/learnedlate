# LearnedLate Wayfinding Visual Prompts

## Core Direction

Chosen direction: wayfinding and navigation.

The visual idea is simple: businesses are trying to find their way through a fast-changing AI landscape. LearnedLate helps them identify the right route, avoid dead ends, and move with more confidence.

This is not a literal map brand. It is a visual language built from:

- routes
- plotted paths
- markers
- signal points
- contour lines
- field notes
- directional overlays

The tone should feel calm, editorial, and sharp. Never futuristic for its own sake.

## Color Behavior

Use the existing dark foundation with amber as the only signal color.

- Backgrounds stay charcoal, graphite, and soft grayscale.
- Amber appears only where attention should go.
- Amber should read as signal, not decoration.
- Do not flood the frame with amber.
- Keep the image mostly desaturated, with one or two amber focal elements.

Amber can be used on:

- route lines
- waypoint dots
- labels
- interface glows
- highlighted edges
- reflected light from a screen or device

## Image Principles

Every image should communicate one of these ideas:

1. Finding the right path.
2. Making sense of a noisy landscape.
3. Turning scattered signals into a clear route.
4. Moving from manual effort to guided systems.

Avoid:

- generic AI neon visuals
- glowing brains
- robot hands
- hologram dashboards floating in space
- cyberpunk scenes
- Greek statues or anything too close to Tenex
- obvious stock office photography

## Art Style

- Editorial, premium, restrained.
- High contrast grayscale base.
- Subtle film grain or soft texture is fine.
- Clean composition with one focal subject.
- Use negative space for headline overlays.
- If people appear, they should feel thoughtful and grounded, not posed.
- If objects appear, prefer tools of orientation: maps, markers, charts, screens, notebooks, desk surfaces, field instruments.

## Homepage Hero Prompt

Intent: urgency plus direction.

Prompt:

Create a premium editorial hero image for an AI consulting website. The theme is wayfinding through change. Show a dark map-like surface or planning desk with a clear plotted route emerging from scattered signals. Mostly grayscale and charcoal tones, with a single amber route line and a few amber waypoint markers as the only strong color. Calm, intelligent, minimal, high-end, spacious composition with negative space for headline text. No generic AI graphics, no robots, no futuristic neon, no people smiling at laptops.

## Strategy Page Hero Prompt

Intent: choosing the right first move.

Prompt:

Create a dark editorial image about choosing the right direction in a complex landscape. Use a topographic map, route sketch, compass-like marker, or plotted decision path. The image should feel focused and analytical, with one amber route or amber decision point standing out against a grayscale background. Premium, minimal, quiet, strategic, with room for headline text. No stock-photo business people. No heavy UI overlays.

## Implementation Page Hero Prompt

Intent: turning messy manual work into a cleaner flow.

Prompt:

Create a premium dark image showing a messy process becoming a clean route. Start with scattered nodes, notes, or fragmented steps on one side and resolve them into one clear path or system line on the other. Grayscale palette with a restrained amber signal line and a few amber markers. Editorial, structured, operational, high contrast, minimal, no cliché AI imagery, no excessive motion blur.

Show a blueprint-like composition, technical sketch, or modular system layout where one custom path or component is highlighted in amber. The image should feel precise, thoughtful, and built for a specific purpose. Mostly grayscale and charcoal, with amber used only for the critical custom element. No sci-fi effects, no glowing code rain, no generic developer desk clutter.

## Shared Section Artwork Prompt

Intent: reusable artwork for cards, breakpoints, or interstitial sections.

Prompt:

Create a minimalist navigation-themed background element for a dark website. Use subtle contour lines, plotted routes, waypoint dots, and directional marks. Keep the composition abstract, elegant, and low-noise. Charcoal and grayscale base with very restrained amber accents. It should feel premium and understated, suitable behind text without overpowering it.

## Video Loop Direction

These should be quiet loops, not promo videos.

Rules:

- Length: 6 to 10 seconds.
- Loop cleanly.
- Motion should be slow and deliberate.
- No rapid cuts.
- No flashy transitions.
- No cinematic trailer energy.

## Homepage Video Loop Prompt

Create a seamless looping background video for a dark AI consulting homepage. Show a subtle map-like surface or abstract route network where faint scattered signals gradually resolve into one clear amber path. Minimal motion, premium editorial style, mostly grayscale with restrained amber highlights, soft texture, quiet and intelligent mood, clean loop, no text.

Create a seamless looping dark video where a messy cluster of faint nodes and lines gradually simplifies into a clean guided workflow path highlighted in amber. The effect should feel operational and elegant, not flashy. Mostly grayscale, restrained amber, no text, no heavy interface graphics.

## Engineering Video Loop Prompt

Create a seamless looping dark video showing a technical schematic or blueprint where one custom component or path lights up in amber and integrates into the full system. Calm, precise, minimal, grayscale base, premium editorial aesthetic, no code rain, no cyberpunk visuals.

## Negative Prompt

Use this as a constraint when generating images or video:

No robots, no humanoid AI, no holograms, no blue neon, no purple gradients, no cyberpunk cityscapes, no generic stock office scenes, no Greek statues, no meme-style visuals, no excessive interface clutter, no cartoonish icons, no glowing brain imagery, no generic SaaS 3D render blobs.

## Composition Notes For Designers

- Leave breathing room for text.
- Keep the main focal element off-center when possible.
- One amber path is stronger than five amber details.
- Images should support copy, not compete with it.
- If in doubt, remove elements rather than add more.
